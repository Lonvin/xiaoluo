# API 选择指南

你的AI依赖多个 API 服务。以下是国内外所有可选方案，按类别排列。

> ⚠️ 所有脚本里的 API key 都是占位符。部署前必须替换成你自己的 key。

---

## 一、大语言模型（核心，必须配）

这是 AI 的「大脑」。所有对话、推理、工具调用都走这里。

### 国内

| 服务 | 模型 | 价格（元/1M tokens） | 推荐理由 |
|------|------|---------------------|---------|
| **DeepSeek** | deepseek-chat | 输入1元 输出2元 | 最便宜的 thinking 模型，中文顶级 |
| **DeepSeek** | deepseek-v4-pro | 输入2元 输出8元 | 我们用的，thinking 模式深度推理 |
| **硅基流动** | DeepSeek-V3/Qwen3 | 输入~1元 输出~2元 | DeepSeek 官方经常崩，这家做备选 |
| **阿里千问** | Qwen3-235B | 输入4元 输出16元 | 中文能力强，有免费额度 |
| **百度文心** | ERNIE 4.5 | 输入4元 输出16元 | 有免费额度 |
| **智谱** | GLM-4-Plus | 输入5元 输出20元 | 中文任务稳定 |
| ** moonshot** | moonshot-v1-128k | 输入8元 输出16元 | 超长上下文（128K） |

### 国际

| 服务 | 模型 | 价格（$/1M tokens） | 推荐理由 |
|------|------|---------------------|---------|
| **OpenAI** | GPT-4o | 输入$2.5 输出$10 | 综合最强 |
| **Anthropic** | Claude Sonnet 4 | 输入$3 输出$15 | 人格表现最佳，写作最像人 |
| **Anthropic** | Claude Opus 4 | 输入$15 输出$75 | 最深度推理，贵 |
| **Google** | Gemini 2.5 Pro | 输入$1.25 输出$10 | 长上下文+免费额度 |
| **xAI** | Grok 4.3 | 输入$2 输出$8 | 实时信息+有个性 |
| **OpenRouter** | 多模型聚合 | 看具体模型 | 一个 key 调所有模型 |

### 怎么选

```
省钱：DeepSeek（国内首选）
人格：Claude Sonnet 4（写作最像人）
稳定：硅基流动 + DeepSeek 模型（DeepSeek 官方崩时备用）
免费额度：千问（Qwen3）、Gemini
```

## 六、微信桥接（必须配）

这是 AI 连接微信的方式。

| 服务 | 说明 |
|------|------|
| **cc-connect + ilink** | 官方微信 Bot API，需要企业认证 |
| **cc-connect + Telegram** | 如果搞不定微信，先用 Telegram 测试 |
| **wx4py**（仅Windows本地） | 操控微信桌面版，不需要企业认证 |

## 七、怎么换 API

所有脚本的 API key 都在文件开头，直接改就行：

```python
# voice_msg.py
TTS_CONFIG = {
    "qwen3-tts-flash": {
        "api_key": "你的千问API_KEY",  # ← 改这里
    }
}

# image_gen.py  
SILICON_KEY = "你的硅基流动API_KEY"  # ← 改这里

# config.toml
system_prompt = "..."  # ← 这里可以指定用哪个模型
```

## 八、推荐组合

### 省钱版（月费 ~20元）
- LLM: DeepSeek
- 搜索: WebSearch（免费）
- 其他: 都不要

### 标准版（月费 ~50元）
- LLM: DeepSeek
- TTS: 千问免费额度
- 图片: 硅基流动
- 搜索: Tavily 免费额度

### 顶配版（月费 ~200元）
- LLM: Claude Sonnet 4（人格最好）
- TTS: ElevenLabs
- 图片: DALL-E 3
- 搜索: Tavily 付费
- 语音识别: Whisper
